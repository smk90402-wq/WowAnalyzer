2026-06-11 야냥 쫄웨이브 분석에서 같은 턴에 3가지 지적받음:
1. **조어/축약 금지**: "톱백"(top100) 같은 임의 축약 쓰지 말 것 → "상위 100" 등 제대로 된 한국어.
   스킬명을 공식 한글 쓰듯, 일반 표현도 자연스러운 한국어로.
2. **WCL cast 이벤트 ≠ 누르는 버튼**: 광포한 야수(1283818)는 패시브 프록 소환(출혈 4%/살명 20%/곰 합류)인데
   cast로 찍혀서 "야격 직후 눌러라"고 잘못 조언함. 사용자가 "패시브 같은데?"로 잡아냄.
   → 스킬을 "누르라"고 말하기 전에 특성트리 노드(PASSIVE/ACTIVE)와 desc 확인. ACTIVE 노드여도
   desc가 "~사용 시 소환"이면 자동(예: 자연의 동맹 노드는 ACTIVE인데 시전 로그 0).
3. **메커니즘 단정은 분리실험 후**: "야격 직격이 휩쓸기로 광역화"는 가이드 문장 재해석+히트수만 보고
   단정한 오류. 단일빌드(난타 미보유) 대조군으로 검증하니 빌드 무관 다중히트(야수별 각 1타)였음.
   → 교란변수 분리 가능한 대조군(다른 빌드/순수단일 보스)을 찾아 검증하고, 못 찾으면 "추정" 명시.

2026-06-12 사격 가이드에서 또 지적받음 (4번째):
4. **"쿨마다" 처방 전에 자원/스택 상호작용 확인**: 파수꾼 속사는 재시전이 총알 세례 스택을 지움 —
   "쿨마다"는 메커니즘 무시 처방. 마무리 사격은 채택률 23%뿐인데 기본 사이클로 서술.
   → 스킬 사용 규칙을 쓰기 전에 ① 그 스킬이 지우는/소모하는 버프 확인 ② 특성 채택률 실측 확인.
   스킬명 자동 아이콘(wsify)은 한글 경계+차단어 필수 — "속사포"가 속사+포로 쪼개졌었음.
