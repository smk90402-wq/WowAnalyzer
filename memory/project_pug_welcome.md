## v3.1 (2026-06-15) — 점수 가중 반영 ★현행★

사용자 요청: 한밤 KR 구인난 심함 → 막공 환영도를 종합 점수에 가중 반영.
**종합 점수 = 파스력(기존 ease·PI·일관성) × 0.65 + 막공환영(pug 1~5→0~1) × 0.35**.
`main.py` `PUG_SCORE_W=0.35` 상수, API에서 블렌딩(파스점수는 score_parse로 보존). 가중치 조정은
이 상수만 고치면 됨. 효과: 사격 9→4·증강 22→12·부정 13→7 상승, 징벌 4→10 하락. 야수 1위 유지.
이전엔 pug가 '점수 미반영 참고 컬럼'이었음. 약어 교정 누적: 용딜=증강, 죽딜=부정/냉죽 무구분(냉죽 4),
혈죽=탱, 스왑힐=하이브리드 원딜(암흑·조화 4).

## v3 (2026-06-15) — 구인 시장 기반 전면 재정의

**사용자 교정(핵심)**: 메타분석은 인원 고정된 정공이 아니라 **막공/신규정공 기준**이어야 함.
"정해진 정규 공격대의 직업분포(킬로그 로스터 TO)보다, 구인 시장에서 중요한 직업이 뭐냐"가 환영지표.
즉 "구인이 어려운(=자리 못 채워 올라오는) 직업 = 환영". 추가 지시: 파티찾기 글은 **본문 탐독**
(제목은 "딜러 구인(세부참조)"뿐, 실제 스펙은 본문 약어), **중복 구인글 제거** 철저히.

**방법(data/pug_demand_inven.json)**: 인벤 board2972 공격대_구인 본문 138개 고유 캠페인
(원본228, 작성자+제목 정규화 중복제거 — 한 글 최대 15회 재등록). #powerbbsContent → 구인 섹션 약어
파서(흑/암/정/냥 단일문자, 죽딜 vs 죽탱 맥락). 두 렌즈: ①구인수요(breadth) 사냥꾼30·증강24·정술21·
죽기20·흑마19 ②수요÷공급(희소+수요=콕집어) 황폐33·증강17·사냥꾼15·정술14 / 징벌1(과잉).

**v2→v3 등급 교정**: 냉법 5→4(법사 구인 9 중위 + 공급과잉 2331 → 신규진입 경쟁적, 정공 1.71은 누적 착시),
황폐 1→4(구인7+공급최소211→수요/공급 33 1위 = 콕집어 환영, 로스터 TO 0.03이 놓침), 포식 5→3(구인 5뿐),
사격·정술 →5. 분포 5:6/4:4/3:4/2:6/1:7. v2 로스터 TO 수치는 팝업 '참고'로 강등.
야수 구인 1위(30/138=22%)=본캐 유리. 점수 미반영 참고 컬럼.

## v2 (2026-06-11) — KR 취업 시장 (로스터 TO) [구버전, v3가 대체]

메타탭 "막공 환영" 컬럼 (`PUG_WELCOME` in app/main.py, 점수 미반영). 2026-06-11 **v2 전면 재설계**.

**★사용자 교정(설계 원칙으로 영구 보존)★**: "많이 보임 ≠ TO". v1이 모집단·top100 고유공대 수로
징벌을 5등급 줬더니 — "정공은 무조건 팔라 한자리 데려가니까 많아 보이는 것뿐, 징기 인구가 많아서
TO는 제일 없는 수준. 제일 막공 취업 안 되는 클". → 개체수 지표는 수요/공급을 혼동한다.
**공급(지원자) 대비 수요(자리수)로 측정해야 함.** 추가 교정 2건: ① "영웅은 아무나 데려가거든"
→ KR 영웅 인구 = 지원자 풀 정의로 사용. ② "직업 자체가 없으면 데려간다 — 공대에 술사가 하나도
없다면 정술인 게 중요" → 클래스 보유율·슬롯 점유 동학 반영.

**v2 방법 (fetch_kr_pug_market.py + analyze_kr_pug_market.py → data/kr_pug_market.json)**:
- 취업률 = KR 신화 유니크 캐릭(27스펙×9보스 전수 — KR은 캡 안 걸림) ÷ KR 영웅 풀(우주의왕관+살라다르; 첫넴은 2000캡이라 제외)
- TO = KR 신화 킬공대 1,316개 로스터(playerDetails, 보스당 150 표본) 공대당 평균 자리수 + 채용%
- 취업률>1 = 보스별 스왑 카드 신호(고통 2.03·사격 1.44·냉죽·파괴 — 본캐가 스펙만 바꿈)
- 무길드 91%라 길드/막공 분리는 신뢰 낮음 (KR 개인로깅 관행)

**핵심 실측**: 징벌 취업률 0.672 하위6위(캡 과대평가)·팔라 보유율 95%·홀팔 782 vs 징벌 940 슬롯 경쟁
→ 2등급. 술사 보유율 100%(전 공대) → 정술 4. 도적 보유율 55%(공대 절반이 도적 0). 냉법 TO 1.71 1위.
등급: 5=냉법·악마·부정·포식·증강 / 4=야수·사격·정술·잠행 / 1=화염·비전·야성·무법·황폐.
