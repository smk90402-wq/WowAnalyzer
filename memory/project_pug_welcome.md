메타탭 "막공 환영" 컬럼 (`PUG_WELCOME` in app/main.py, 점수 미반영). 2026-06-11 **v2 전면 재설계**.

**★사용자 교정(설계 원칙으로 영구 보존)★**: "많이 보임 ≠ TO". v1이 모집단·top100 고유공대 수로
징벌을 5등급 줬더니 — "정공은 무조건 팔라 한자리 데려가니까 많아 보이는 것뿐, 징기 인구가 많아서
TO는 제일 없는 수준. 제일 막공 취업 안 되는 클". → 개체수 지표는 수요/공급을 혼동한다.
**공급(지원자) 대비 수요(자리수)로 측정해야 함.** 추가 교정 2건: ① "영웅은 아무나 데려가거든"
→ KR 영웅 인구 = 지원자 풀 정의로 사용. ② "직업 자체가 없으면 데려간다 — 공대에 술사가 하나도
없다면 정술인 게 중요" → 클래스 보유율·슬롯 점유 동학 반영.

**v2 방법 (fetch_kr_pug_market.py + analyze_kr_pug_market.py → data/kr_pug_market.json)**:
- 취업률 = KR 신화 유니크 캐릭(27스펙×9보스 전수 — KR은 캡 안 걸림) ÷ KR 영웅 풀(우주의왕관+살라다르; 첫넴은 2000캡이라 제외)
- TO = KR 신화 킬공대 1,316개 로스터(playerDetails, 보스당 150 표본) 공대당 평균 자리수 + 채용%
- 취업률>1 = 보스별 스왑 카드 신호(고통 2.03·사격 1.44·냉죽·파괴 — 본캐가 스펙만 바꿈)
- 무길드 91%라 길드/막공 분리는 신뢰 낮음 (KR 개인로깅 관행)

**핵심 실측**: 징벌 취업률 0.672 하위6위(캡 과대평가)·팔라 보유율 95%·홀팔 782 vs 징벌 940 슬롯 경쟁
→ 2등급. 술사 보유율 100%(전 공대) → 정술 4. 도적 보유율 55%(공대 절반이 도적 0). 냉법 TO 1.71 1위.
