# 사내망 웹 배포 — 피드백 기능 완료 후 착수

사용자 요청(2026-06-13): 회사 사내망(각 PC에 IP 할당, 인터넷 가능)에서 **브라우저로 접근 가능한 웹페이지**로. 공개 인터넷 퍼블리시는 복잡하니 **안 함** — LAN 내 `http://<호스트IP>:<포트>` 접속.

**현재 상태:** serve.py = FastAPI + pywebview 데스크톱 앱, uvicorn이 `127.0.0.1`(로컬만) 바인딩(test.err 로그상 포트 9896 류). 인증(Phase 1: ID/PW + 서명 세션쿠키, 커밋 eccf33f) 이미 있음.

**필요 작업:**
1. uvicorn 바인딩 `127.0.0.1` → `0.0.0.0`.
2. 고정 포트 결정.
3. Windows 방화벽 인바운드 규칙(그 포트 TCP) 허용.
4. 호스트 PC LAN IP 확인(ipconfig) → 동료는 `http://<IP>:<포트>`.
5. (권장) 헤드리스 모드: pywebview 창 없이 서버만 도는 실행 경로 추가 → 한 PC를 상시 서버로.
6. 데이터 캐시(LFS 대용량)는 호스트 PC에 존재해야 함.

**착수 시 결정 3개:** ① 상시 호스트 PC ② 헤드리스 분리 여부 ③ 포트.
**보안:** LAN 한정 + 기존 인증 → 합리적. 공개 노출 아님.
